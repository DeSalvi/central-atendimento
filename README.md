Sistema de Central de Atendimento

Projeto desenvolvido para a disciplina de Estruturas de Dados.

📌 Descrição

Sistema para gerenciamento de processos em uma central de atendimento, com controle de prioridade e histórico de atendimentos.

⚙️ Funcionalidades
Cadastro de processos
Geração automática de protocolo
Armazenamento em vetor dinâmico
Pilha de atendimento (LIFO)
Desfazer último atendimento (undo)
Atendimento por prioridade

🛠️ Tecnologias
Java

📂 Estrutura
model → entidades do sistema
estrutura → estruturas de dados
service → regras de negócio
app → execução do sistema
